# Cursor Frontend Clone

A pixel-perfect recreation of the Cursor website landing page.

## Sections Recreated

- **Navigation Bar**: Includes the logo, navigation links (Product, Enterprise, Pricing, Resources), and action buttons (Sign in, Download).
- **Hero Section**: Features the main headline, a call-to-action button, and a hero image demonstrating the editor.
- **Company Section**: A "Trusted by" section displaying logos of world-class software teams.
- **Feature Cards**: Detailed feature breakdown with alternating image alignments and clear call-to-actions.

## Fonts and Colors Used

### Fonts
- **CursorGothic**: Used throughout the application for headlines and body text to match the original branding.

### Colors
- **Background**: `#15120B` (Primary Dark)
- **Primary Accent**: `#BF420F` (Orange/Rust)
- **Secondary Accent**: `#EEECEC` (Off-white)
- **Text Secondary**: `#999997` (Gray)
